## Город внутри тебя

Исследование эмоционального ландшафта Петербурга.
Проект-навигатор по внутреннему миру, скрытый в фасадах дворцов и строгих линиях заводов. Петербург — это масштабная проекция человеческой души через холодный гранит и строгие чертежи.

---

## О проекте

Мы разложили код города на четыре базовых чувства:

- **Радость** — Елизаветинское барокко (Петергоф, Смольный собор, Зимний дворец)
- **Спокойствие** — Русский классицизм (Исаакиевский собор, Казанский собор, Таврический дворец)
- **Тайна** — Северный модерн (Дом Зингера, дом Лихачёва, дом Бака)
- **Стремление** — Ленинградский конструктивизм (ДК связи, Левашовский хлебозавод, водонапорная башня)

Каждое чувство раскрывается через три архитектурных памятника с уникальными текстами и визуальным рядом.

## Технологии

- HTML5 / CSS3
- JavaScript (нативная сборка, без фреймворков)
- GitHub Pages

### Локально
1. Клонировать репозиторий git clone https://github.com/gina-eifman/city-inside.git
2. Открыть папку проекта cd city-inside
